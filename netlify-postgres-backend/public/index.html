Ukraine Experts Database API

This is the API server for the Ukraine Experts Database. The API provides access to experts, organizations, and related data.

Available Endpoints:

GET /api/ - Welcome message and API info

GET /api/experts - List all experts

POST /api/experts - Create a new expert

GET /api/experts/{id} - Get expert details

PUT /api/experts/{id} - Update an expert

DELETE /api/experts/{id} - Delete an expert

GET /api/cities - List all cities

GET /api/search?q={term} - Search for experts

GET /api/statistics - Get database statistics
